# PhikozzLibrary.Pooling - ObjectPool

- **ObjectPool\<T\>**은 오브젝트의 재사용을 위한 범용 오브젝트 풀 클래스입니다.  
  오브젝트의 생성, 반환, 초기화 시 커스텀 동작을 지정할 수 있습니다.

## IObjectPool\<T\> (interface)

- 오브젝트 풀의 기본 동작을 정의하는 인터페이스입니다.

| 메서드         | 설명                       |
| -------------- | -------------------------- |
| `Get()`        | 오브젝트를 풀에서 꺼냅니다. |
| `Set(obj)` | 오브젝트를 풀에 반환합니다. |
| `Clear()`      | 풀을 비웁니다.             |

## ObjectPool\<T\> (class)

- 오브젝트 풀의 기본 구현체입니다.

### 생성자

- **ObjectPool(Func\<T\> createFunc, Action\<T\> onGet = null, Action\<T\> onSet = null)**
  - `createFunc`: 오브젝트 생성 함수 (필수)
  - `onGet`: 오브젝트 꺼낼 때 실행할 액션 (선택)
  - `onSet`: 오브젝트 반환 시 실행할 액션 (선택)

### 주요 메서드

- **Get()**:  
  풀에 오브젝트가 있으면 꺼내고, 없으면 새로 생성합니다.  
  꺼낼 때 `onGet` 액션이 실행됩니다.

- **Set(obj)**:  
  오브젝트를 풀에 넣습니다.  
  반환 시 `onSet` 액션이 실행됩니다.

- **Clear()**:  
  풀에 있는 모든 오브젝트를 제거합니다.

---

> 오브젝트 풀을 사용하면 반복적으로 생성/파괴되는 오브젝트의 성능을 최적화할 수 있습니다.  
> `ObjectPool<T>`는 다양한 타입에 대해 재사용이 가능합니다.
